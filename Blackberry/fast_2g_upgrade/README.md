# Fast 2.4GHz Wi-Fi Upgrade — One-Click GitHub Version

Improve your 2.4GHz Wi-Fi for faster speeds, lower latency, and higher reliability.

## Quick Start
1. Copy all files to your system.
2. Edit `hostapd.conf` with your SSID and passphrase.
3. Run `tune-network.sh` as root.
4. (Optional) Enable the systemd service for auto-tuning.
